# Portfolio

Personal developer portfolio showcasing selected projects and professional experience.

## Overview
This repository contains my personal developer portfolio.
It presents a curated selection of projects, technical skills, and experience.
The website is built as a statically generated application using Next.js.

## Tech Stack
- Next.js
- React
- TypeScript
- Tailwind CSS
- MDX

## Getting Started
npm install  
npm run dev

## Deployment
Static export deployed via GitHub Pages.
