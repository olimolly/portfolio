## Overview

- Development of a B2B application for an internet service provider.
- Integration and consumption of external provider APIs using a Laravel back-end architecture.
- Implementation of application middlewares to manage service-to-service data flows.
- Use of Redis caching to improve performance and response times.
- Data storage and management with PostgreSQL.
- Ongoing API maintenance and inter-service communication improvements.
